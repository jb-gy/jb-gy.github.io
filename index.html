Stock Analysis Engine

Python NumPy SciPy Matplotlib Seaborn Yfinance

This Python-based tool analyzes any publicly traded stock using end-of-day data from Yahoo Finance. It computes key financial metrics such as historical volatility, skewness, kurtosis, Sharpe Ratio, and annualized returns. Users can visualize returns distributions (with a normal distribution comparison), rolling volatility, and multi-stock comparative statistics. The tool also features efficient data caching to streamline repeated analyses.

PlanMyCash - HackwesTX group project

TypeScript Python Plaid API Terraform AWS API gateway AWS Lambda SQL Alchemy

This budget planning web app lets users track expenses, manage budgets, and visualize spending patterns in real time.
